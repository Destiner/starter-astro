# Astro Starter

(Almost) unopinionated, unstyled Astro starter, useful for quick prototypes and theme creation.

## Getting Started

```bash
npm i
npm run dev
```

## Features

* Mobile-first
* Better typography
* Dark mode
* Images support
* Analytics
* SEO
* Favicon
* Vue integration

## DevEx 

* ESLint
* Prettier
* Stylelint
* TS configuration
* CI (via GH actions)

## Customization

* Fonts (Google Fonts)
* Colors (CSS variables)
* Analytics (`astro-analytics`)
