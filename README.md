# Astro Starter

(Almost) unopinionated, unstyled Astro starter, useful for quick prototypes and theme creation.

## Getting Started

```bash
npm i
npm run dev
```

## Features

* Mobile-first
* Typography tweaks
* Dark mode
* Images support
* Analytics (via `astro-analytics`)
* SEO
* Favicon
* Vue integration

## DevEx 

* ESLint
* Prettier
* Stylelint
* TS configuration
* CI (via GH actions)

## Customization

* Change fonts
* Change or remove analytics
